Projects:

  • Personal Projects
    • Mezcladorme View Mezcladorme live here. A website based off of Spotify’s “Sort Your Music” that loads a user’s Spotify playlists and rearranges the song order by creating a minimum spanning tree with edge weights based off of song similarity. The similarity between songs is found by comparing beat, volume, and other audio features. Originally written in jQuery for BrickHack 2016 with plans to move more of the source code to the Django backend. Source code can be found here on GitHub.
    • Modular Multiplicative Inverse Calculator View the calculator live here. A script to calculate and show the equations used to find the GCD then multiplicative inverse (if there is one) for a number in a modular ring up to 50,000,000. Finding the modular multiplicative inverse is used in practical applications to speed up the encryption process of RSA. Thanks to Adam McCarthy for helping test the script. View the source code here on GitHub.

  • Work Related Projects
    • Connecting Hudl Connecting Hudl was a project as part of Hudl's "Mind the Gap" week, a more product/process oriented version of Skunkworks aimed towards fixing gaps in Hudl and the Hudl project. The Connecting Hudl team included Chastity Grant, Mike Jensen, and Austin Wendt and had three parts: a Hudl internal Chrome extension, internal "role" pages, and an internal Wiki. The Chrome extension allowed the user to pick a role ("Developer", "Designer", "QA", etc) and view a small extension window of popular links for that role along with company useful internal links. The extension stored the user's role for later visits but still allowed the user to view different pages for different roles. The internal role pages displayed photos for all Hudlies working in that role along with a master list of tools and documentation needed for that role. The internal wiki was an installation of MediaWiki and during Mind the Gap the team created/wrote many companywide reaching articles. Connecting Hudl won the 2015 Mind the Gap "Best in Show" award, chosen by Hudlie vote.
